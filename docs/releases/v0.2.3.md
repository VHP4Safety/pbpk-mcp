# PBPK MCP v0.2.3

Suggested GitHub release title:

- `PBPK MCP v0.2.3: version-alignment patch`

Suggested short release summary:

- PBPK MCP `v0.2.3` is a patch release that aligns the repository and runtime version markers after the `v0.2.2` publication, so the tagged release and the visible project metadata remain consistent.

## Highlights

- aligns `README` current-release labeling with the latest published patch release
- aligns compose/runtime `SERVICE_VERSION`
- aligns package metadata and API/OpenAPI version markers in the clean GitHub repo
- keeps the `v0.2.2` OECD-reporting feature set intact while making the release surfaces consistent

## Included In This Release

### Release-surface alignment

- README current release marker updated to `v0.2.3`
- changelog updated with an explicit `v0.2.3` entry
- compose/runtime version marker advanced to `0.2.3`
- package metadata and OpenAPI version markers aligned in the published repo

## Limitations To Call Out Publicly

- this is a release-alignment patch, not a new feature release
- Berkeley Madonna `.mmd` is still a conversion source, not a runtime backend
- runtime success and dossier export do not automatically imply regulatory qualification
